# ace-digit

FrontEnd + BackEnd

## Requirement
- Make sure node v16+

## Project configuration
- Frontend: React, javascript
- Backend: NodeJS, typescript, postgresql, express, sequelize


## Installation.
### Front End.
- `cd frontend`
- `npm install`
- `npm run start`

### Back End.
- `cd backend`
- `npm install`
- `npm run start`

## API
- http://localhost:8001/

## REST API Endpoint
Use API endpoint in Postman or other API management tools by using the below file.
`api-data.json`

## FrontEnd Url
- http://localhost:3000/
